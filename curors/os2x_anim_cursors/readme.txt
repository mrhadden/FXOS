Workbench 2.x/3.x Animated Cursors for Windows 9x/ME
----------------------------------------------------------------
These are mouse pointer replacements for Windows, based on Amiga
Workbench versions 2 and 3.  The original cursors are by P. Moeckli.
Animated versions by me.


Installation:
----------------------------------------------------------------
Simply unzip this archive to your Windows/cursors folder and you are 
ready to go!
